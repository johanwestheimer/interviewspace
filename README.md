# interviewspace

Подготовка к техническим собеседованиям. Структурированные курсы «с нуля до собеседования» с понятными уроками, конспектами и тестами.

## Содержание

### 💻 Computer Networks (Компьютерные сети)

Полный курс по компьютерным сетям с полного нуля до уровня, достаточного для прохождения собеседований в ведущие компании.

- 📍 [`computer/networks/`](computer/networks/)
- 🗺️ **План курса и конспекты:** [`computer/networks/lessons/main.md`](computer/networks/lessons/main.md)
- 📚 **Уроки:** [`computer/networks/lessons/`](computer/networks/lessons/)

Структура курса: **9 частей → 47 уроков**, в конце каждой части — тест для самопроверки. Язык объяснений — русский, примеры кода — на C++.

## Как пользоваться

1. Открой `main.md` нужного курса — это оглавление и шпаргалка: по каждому уроку есть краткий конспект и список того, что обязательно выучить.
2. Проходи уроки по порядку: они написаны как единый связный рассказ, где каждый следующий опирается на предыдущий.
3. В конце части реши тест, прежде чем двигаться дальше.
